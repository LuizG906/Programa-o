A tag input cria campos de entrada de dados em formulários HTML,

enquanto a tag label define o rótulo para um campo de entrada,

tornando o formulário mais acessível e fácil de usar.

A Tag input:

- Cria diferentes tipos de campos de entrada, como caixas de texto, caixas de seleção, botões de rádio, seleções, etc.

- É usada dentro da tag

para coletar informações do usuário.

- O atributo type define o tipo de campo de entrada (por exemplo, text, email, date, checkbox).
