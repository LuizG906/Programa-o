Conceitos de DIV,H1 e H2:

No HTML, as tags "div" agrupam blocos de elementos, enquanto as tags "h1" e "h2" são usadas para criar uma hierarquia de títulos e subtítulos

Laços de repetição:

Em programação, laços de repetição, também conhecidos como loops ou laços de iteração, são estruturas que executam um bloco de código várias vezes. São úteis para automatizar tarefas que precisam ser repetidas.

Linguagens de programação:

Linguagem de programação é um conjunto de regras que permite criar instruções para um computador. Com ela, é possível desenvolver programas, aplicativos e outras soluções tecnológicas.

Ex: HTML/CSS, Go, Ruby, PHP, ou Python

Principais TAGS:

As principais tags HTML são as que definem a estrutura e o conteúdo de uma página web. Algumas das tags mais comuns são:

p: Define um parágrafo

h1: Define um título de maior nível hierárquico

h2: Define um título de segundo nível hierárquico

h3: Define um título de terceiro nível hierárquico

br: Insere uma quebra de linha

hr: Define um filete horizontal

!--: Define um comentário

head: Define o cabeçalho principal do documento

form: Cria um formulário para entrada de dados

input: Cria campos de entrada em formulários

Background-color:

A propriedade background-color do CSS define a cor de fundo de um elemento HTML. É possível usar essa propriedade para definir a cor de fundo de uma página web ou de um elemento específico.

Fonte-size:

Font size é uma expressão em inglês que significa "tamanho da fonte" ou "tamanho de letra"

No HTML, pode usar o elemento (big) para aumentar o tamanho do texto.

No CSS, pode usar a propriedade font-size e atribuir-lhe um valor.

No JavaScript, pode usar o método fontsize().

No Microsoft Word, pode usar a propriedade Font.Size

text-aling:

"Text align" é uma propriedade que define como o texto está alinhado, seja em controles, parágrafos ou elementos HTML.

Text-transform:

A propriedade text-transform do CSS define como as letras de um texto serão exibidas na tela. Por exemplo, você pode usar text-transform para deixar todas as letras de um texto em maiúsculas ou minúsculas.

Console:

O que é? Para que serve?

Em programação, um console é uma interface de linha de comando que permite a interação com um sistema computacional. Ele é usado para executar comandos, exibir mensagens e registrar dados.
